<a id='x7-60006'></a>Crecimiento y decrecimiento de una función

Se dice que una función f es creciente en el intervalo I si para todo x,y ∈ I

x < y⇒f(x) ≤ f(y)

Asimismo, se dice que f es decreciente en I si para todo x,y ∈ I

x < y⇒f(x) ≥ f(y)

Una función se llama monótona en un intervalo cuando es creciente o decreciente en dicho intervalo.

La figura siguiente sugiere que si la gráfica de una función derivable tiene las rectas tangentes con pendientes positivas en un intervalo, entonces la función es creciente en el intervalo. Análogamente, si la gráfica de una función tiene las rectas tangentes con pendientes negativas, entonces la función es decreciente en el intervalo.

Esta idea intuitiva queda recogida en el siguiente resultado.

Teorema 3 Si f es derivable en (a,b), entonces: (1) f es creciente en (a,b) si y sólo si f′(x) ≥ 0 para todo x ∈ (a,b), y, (2) f es decreciente en (a,b) si y sólo si f′(x) ≤ 0 para todo x ∈ (a,b).

Demostración: (1) Supongamos que f′(x) ≥ 0 para toda x ∈ (a,b). Si x y y son dos puntos cualesquiera de (a,b) con x < y, entonces aplicamos el teorema del valor medio a f en el intervalo cerrado [x,y] para obtener un punto c del intervalo (x,y) tal que
f(y) − f(x) = f′(c) ⋅ (y − x)

Puesto que por hipótesis f′(c) ≥ 0 y además se tiene y − x > 0, se sigue que f(y) − f(x) ≥ 0. Por tanto, f(x) ≤ f(y) y, como x y y son dos puntos cualesquiera del intervalo, se concluye que f es creciente en (a,b). Recíprocamente, supongamos que f es derivable y creciente en (a,b). Consideremos un punto cualquiera c de (a,b), entonces para todo x ∈ (a,b) tal que x > c, tenemos

f(x) − f(c) x − c ≥ 0

Del mismo modo, para todo x ∈ (a,b) tal que x < c, tenemos

f(x) − f(c) x − c ≥ 0

Entonces, por las propiedades de los límites y la definición de derivada, tenemos

f′(c) = lim ⁡ x→cf(x) − f(c) x − c ≥ 0

que es lo que queríamos demostrar. (2) La demostración es similar y se deja al lector. □

Se dice que una función f es estrictamente creciente en el intervalo I si para todo x,y ∈ I

x < y⇒f(x) < f(y)

Análogamente, se dice que f es estrictamente  decreciente en I si para todo x,y ∈ I

x < y⇒f(x) > f(y)

Una función se llama monótona estrictamente en un intervalo si la función es estrictamente creciente o decreciente en dicho intervalo.

La siguiente figura muestra que si una función tiene derivada positiva en un intervalo, entonces la función es estrictamente creciente en este intervalo. Análogamente, si la función tiene derivada negativa en un intervalo, entonces es estrictamente decreciente en el intervalo. Mientras que si la derivada es nula, entonces la función es evidentemente constante en el intervalo.

Esta idea intuitiva queda recogida en el siguiente resultado.

Teorema 4 Si f es derivable en el intervalo (a,b), entonces: (1) si f′(x) > 0 para todo x ∈ (a,b), entonces f es estrictamente creciente, y, (2) si f′(x) < 0 para todo x ∈ (a,b), entonces f es estrictamente decreciente.

Demostración: La demostración de este hecho es similar a la prueba del teorema anterior y se deja para el lector. □

El recíproco del teorema 4 no se cumple en general, pues una función derivable estrictamente creciente puede tener derivada nula en determinados puntos. Por ejemplo, la función f(x) = x3 es estrictamente creciente en todo ℝ y, en cambio, se cumple f′(0) = 0.

La situación es similar para las funciones estrictamente decrecientes.

Es importante subrayar que de las propiedades anteriores no se deduce que, si una función tiene derivada positiva o negativa en un punto, entonces la función es creciente o decreciente en este punto, pues, por ejemplo, la siguiente función

Por lo tanto, para que una función sea creciente o decreciente en un punto es preciso que la función sea creciente o decreciente en un intervalo abierto que contiene al punto. Además, si la función es derivable, será creciente o decreciente si la derivada es positiva o negativa en el intervalo.

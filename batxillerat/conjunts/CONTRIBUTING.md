# Com contribuir

Gràcies per voler millorar aquest material.

## Propostes útils

- Correccions lingüístiques o tipogràfiques.
- Correccions matemàtiques.
- Millores en les respostes del test.
- Nous tests o exercicis sobre conjunts, relacions i aplicacions.
- Millores en la compatibilitat del PDF interactiu.

## Recomanacions

- Feu canvis petits i ben delimitats.
- Si corregiu una resposta del test, expliqueu breument el motiu matemàtic.
- Comproveu, si és possible, que el document compila amb:

```bash
latexmk -pdf -interaction=nonstopmode -halt-on-error main.tex
```

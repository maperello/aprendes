# Projecte de Lògica — Batxillerat

Aquest projecte conté els materials de l’assignatura de Lògica per a Batxillerat avançat.  
Inclou:

- Estructura modular en LaTeX
- Capítols independents
- Preàmbul compartit
- Figures i diagrames
- PDF final generat

## Estructura del projecte

batxillerat/logic/
├── preamble/
├── chapters/
├── figures/
└── pdf/


## Compilació

El fitxer principal serà:

main.tex


La compilació recomanada és:

- `pdflatex`
- `biber` (si fem servir bibliografia)
- `pdflatex` dues vegades

## Objectiu

Crear un llibre de lògica clar, modular i adaptat a estudiants de Batxillerat amb interès avançat en matemàtiques.

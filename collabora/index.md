# Col·labora amb **aprendes**

**aprendes** és un projecte obert de materials docents de matemàtiques per a Batxillerat i Universitat.

Els documents estan escrits en LaTeX i es poden millorar mitjançant GitHub.

## Si ets alumne

Pots ajudar indicant:

- errors tipogràfics;
- exercicis que no s’entenen;
- passos massa ràpids;
- solucions incorrectes;
- apartats on caldrien més exemples.

No cal saber LaTeX per informar d’un error o proposar una millora.

## Si ets professor

Pots contribuir proposant:

- millores didàctiques;
- nous exercicis;
- correccions matemàtiques;
- canvis en l’ordre d’exposició;
- adaptacions al currículum;
- observacions sobre el nivell de dificultat.

## Si coneixes LaTeX

També pots contribuir directament modificant els fitxers `.tex` i enviant una pull request.

El repositori de GitHub és l’espai on es gestionen les propostes, errors i millores dels documents.

[Anar al repositori de GitHub](https://github.com/maperello/aprendes)
